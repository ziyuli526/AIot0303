# ⏰ Ziyu Li — Personal Clock Page
file:///C:/%E7%89%A9%E8%81%AF%E7%B6%B2/L1/index.html
A beautifully designed personal landing page featuring a **real-time live clock** with Taipei Standard Time (UTC+08:00).

## ✨ Features

- **Live Digital Clock** — Hours, minutes, and seconds update in real time with smooth flip animations
- **Minute Progress Bar** — Visual indicator showing the progress within the current minute
- **Date & Time Info** — Displays full date, day of the year, week number, day of the week, and year progress
- **Modern Dark UI** — Sleek glassmorphism design with animated gradient orbs and noise texture background
- **Responsive Layout** — Fully responsive across desktop, tablet, and mobile devices
- **Smooth Animations** — Scroll-triggered reveal animations, blinking separators, and heartbeat effects

## 🛠 Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, gradients, glassmorphism, keyframe animations, responsive media queries
- **Vanilla JavaScript** — Real-time clock logic, scroll-based reveal animations
- **Google Fonts** — [Outfit](https://fonts.google.com/specimen/Outfit) & [Space Mono](https://fonts.google.com/specimen/Space+Mono)

## 🚀 Getting Started

Simply open `index.html` in any modern browser — no build tools or dependencies required.

```bash
# Clone the repository
git clone https://github.com/ziyuli526/AIot0303.git

# Open in browser
start index.html
```
